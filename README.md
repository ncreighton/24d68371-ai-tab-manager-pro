# AI Tab Manager Pro

AI Tab Manager Pro is a smart Chrome browser extension that automatically organizes your tabs using AI-powered topic detection. Stop losing track of important research, work projects, and reading lists. The extension identifies what each tab is about and intelligently groups related tabs together, monitors your memory usage in real time, and lets you archive entire tab sessions with a single click. When you come back to a project, restore your full working context instantly. Built for power users, developers, researchers, and anyone managing 10+ tabs daily. Works entirely in your browser — no data sent to external servers.

## Installation

### Chrome
1. Download and extract the ZIP
2. Go to `chrome://extensions/`
3. Enable "Developer mode"
4. Click "Load unpacked" and select the extracted folder

### Firefox
1. Download the Firefox version
2. Go to `about:debugging`
3. Click "Load Temporary Add-on"
